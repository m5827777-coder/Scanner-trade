Загрузка стратегий
Глобальные параметры бота
positionSize — Размер позиции $
Сумма в долларах на каждую сделку
cooldownMin — Cooldown (минуты)
Минимальный интервал перед повторным входом по токену+стратегии
maxOpenPos — Макс. открытых позиций
При достижении лимита бот не открывает новые позиции
Как работает сохранение параметров
Параметры хранятся в файле data/params.json в вашем репозитории. Бот читает их при каждом запуске. Чтобы сохранять изменения прямо из этого дашборда, введите GitHub Token с правами repo.
Где взять GitHub Token:
github.com → Settings → Developer settings → Personal access tokens → Tokens (classic) → Generate new token → выберите scope: repo → Generate → скопируйте
